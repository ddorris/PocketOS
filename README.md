# PocketOS

PocketOS is a tiny platform for personal games and apps.

- Cost free
- Ad free
- No account
- No purchases
- No wasted time

## Tech Stack

- **Language:** JavaScript
- **Graphics Library:** p5.js

## Development

This project is developed using VS Code and the Live Server extension. This allows for live testing on both desktop (Chrome with DevTools) and mobile devices (iPhone).
